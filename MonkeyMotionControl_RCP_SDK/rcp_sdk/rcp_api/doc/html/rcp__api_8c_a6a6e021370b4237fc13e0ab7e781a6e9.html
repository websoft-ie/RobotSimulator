◆ RCP_API_ENABLE_STR_TO_ENUM

#define RCP_API_ENABLE_STR_TO_ENUM

If disabled, there will be no support for converting parameter ids (rcp_param_t) to strings, or vice versa. rcp_get_name will always return NULL and rcp_get_id will always return RCP_PARAM_COUNT
